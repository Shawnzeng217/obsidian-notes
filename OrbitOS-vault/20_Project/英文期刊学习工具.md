# 英文期刊学习工具

## 📌 项目概述

**目标**：创建一个集成翻译、学习追踪的英文期刊阅读与学习工具，帮助通过定期订阅的英文期刊进行英语学习。

**核心价值**：
- 📖 便捷的期刊管理与阅读体验
- 🔄 自动化翻译辅助
- 📚 生词积累与学习追踪
- 📊 学习进度可视化

**当前状态**：需求分析 → 系统设计 → 原型开发

---

## 🎯 主要需求

### 1. 期刊源管理
- [ ] **期刊来源识别**
  - 格式支持：PDF、HTML 邮件、Web 链接、纯文本
  - 源类型：订阅邮件、在线平台、RSS Feed
  - 更新频率：不定时，待统计分析

- [ ] **自动入库流程**
  - 邮件自动检测和导入
  - 链接快速导入
  - 批量处理机制

### 2. 阅读与翻译
- [ ] **划词翻译**
  - 句子级翻译
  - 单词 / 短语查询
  - 快速标注

- [ ] **翻译策略**
  - 全文翻译（初读）
  - 按段落翻译（深入理解）
  - 按需翻译（精读模式）

- [ ] **翻译集成**
  - 集成 [translate-book](https://github.com/deusyu/translate-book/tree/main) 工具
  - 支持多种翻译 API（Google Translate、Claude API 等）
  - 保留原文与翻译对照

### 3. 英语学习追踪
- [ ] **生词管理**
  - 自动提取 / 手动标记生词
  - 生词本（按文章、主题分类）
  - 单词例句保存

- [ ] **难点标记**
  - 标记困难表达、语法点
  - 短语搭配记录
  - 文化背景注解

- [ ] **学习统计**
  - 阅读统计（篇数、字数、时间）
  - 生词学习进度
  - 复习计划与提醒

### 4. 系统集成
- [ ] **Obsidian 集成**
  - Markdown 存储格式
  - Wiki-links 内部链接
  - Dataview 动态查询
  - 标签与元数据管理

- [ ] **自动化脚本**
  - 定期同步更新
  - 批量处理
  - 数据备份

## 📋 核心功能模块

### Module 1: 期刊管理
**功能**：期刊的导入、分类、检索

```yaml
# 期刊元数据结构
title: "[期刊名] 2026-03"
source: "邮件 / Web / RSS"
date_received: 2026-03-15
category: "科技 / 商业 / 科学"
articles: 5
word_count: 5000
status: "待读 / 已读 / 复习中"
tags: [期刊, AI, 科技]
```

### Module 2: 翻译系统
**功能**：集成 translate-book，支持多粒度翻译

```
原文段落
↓
[translate-book CLI / API]
↓
中英对照版本
↓
Markdown 存储
```

### Module 3: 生词库
**功能**：自动/手动提取生词，构建个人词库

```yaml
# 生词卡模板
word: "serendipity"
translation: "幸运巧合"
part_of_speech: "noun"
example_sentence: "..."
context_article: "[[2026-03-期刊名]]"
learned_date: 2026-03-16
review_count: 0
mastery: "新学"
```

### Module 4: 学习追踪
**功能**：记录学习进度，生成统计数据

```yaml
# 学习日志
date: 2026-03-16
articles_read: 1
new_words: 12
review_count: 5
total_learning_time: 45min
focus_areas: ["技术术语", "商业用语"]
```

### Module 5: 复习系统
**功能**：间隔重复式复习（Spaced Repetition）

- 根据学习历史自动生成复习任务
- 优先复习低掌握度词汇
- 追踪复习进度

## 🛠️ translate-book 集成方案

**项目地址**：https://github.com/deusyu/translate-book

### 集成方式

#### Option A: CLI 集成（推荐）
```bash
# 安装 translate-book
npm install -g translate-book

# 翻译单个文件
translate-book translate input.md -o output.md

# 批量翻译
translate-book batch ./articles/ -o ./translated/
```

#### Option B: Node.js 库集成
```javascript
const { TranslateBook } = require('translate-book');

const translator = new TranslateBook({
  apiKey: process.env.TRANSLATE_API_KEY,
  provider: 'google' // or 'claude'
});

const result = await translator.translate(text);
```

#### Option C: 工作流脚本
创建 Node.js 脚本处理：
1. 监听新期刊
2. 自动调用翻译
3. 生成对照版本
4. 更新 Obsidian
